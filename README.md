# Hope Speech Detection in Code-Mixed Tulu Language

## 📌 Project Overview

This project detects Hope Speech in Code-Mixed Tulu-English text using Machine Learning and Natural Language Processing (NLP).

The system classifies social media text into hope speech and non-hope speech categories after preprocessing and feature extraction.

## 🚀 Features

- Text preprocessing
- TF-IDF feature extraction
- Logistic Regression
- Support Vector Machine (SVM)
- Random Forest
- Naive Bayes
- Voting Classifier

## 🛠 Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- NLTK

## 📊 Model Performance

| Model | Accuracy |
|--------|----------|
| SVM | 67.29% |
| Random Forest | 65.96% |
| Naive Bayes | 65.58% |
| Voting Classifier | 65.11% |
| Logistic Regression | 63.38% |
